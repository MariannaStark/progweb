Mini Tutorial HTML

Tags Estruturais

<b> Apenas deixa o texto em negrito (sem semântica).
Exemplo: <p>Texto em <b>negrito</b></p> - efeito: negrito

<i> Apenas deixa o texto em itálico (sem semântica).
Exemplo: <p>Texto em <i>itálico</i></p> - efeito: itálico

<u> Sublinha o texto, sem indicar significado semântico.
Exemplo: <p>Texto <u>sublinhado</u></p> - efeito: sublinhado

<br> Quebra de linha
Exemplo: quebra de linha<br>próxima linha

<hr> Linha horizontal (separa visualmente ).
Exemplo: <hr>

<span> Usado para aplicar estilo inline, sem significado semântico.
Exemplo: <span>destaque</span>

Tags Semânticas

<strong> Dá ênfase forte (semântica de importância).
Exemplo: <strong>Importante</strong> - efeito: Importante

<em> Dá ênfase leve (semântica de entonação).
Exemplo: <em>Leve ênfase</em> - efeito: Leve ênfase

<blockquote> Indica uma citação longa.
Exemplo: <blockquote>Citação longa</blockquote> - efeito:

Marianna Stark

<q> Indica uma citação curta (inline).
Exemplo: <q>Citação curta</q> - efeito: Citação curta

<abbr> Define uma abreviação.
Exemplo: <abbr title="Hypertext Markup Language">HTML</abbr> - efeito: HTML

<cite> Define a fonte de uma citação (ex: nome de um livro, filme).
Exemplo: <cite>Uma Linda mulher</cite> - efeito: Uma linda mulher

<dfn> Marca a definição de um termo.
Exemplo: <dfn>HTML</dfn> - efeito: HTML

<address>Informações de contato (autor, organização).
Exemplo: <address>Marianna- Serra, Brasil</address> - efeito:

Marianna - Serra, Brasil

<sup> Indica expoentes.
Exemplo: 9<sup>2</sup> - efeito: 92

<sub> Indica conteúdo inferior (ex: fórmulas químicas).
Exemplo: H<sub>2</sub>O - efeito: H2O
